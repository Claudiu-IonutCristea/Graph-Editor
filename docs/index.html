Force Mode

Forces are applied on the nodes of the graph in order to arrange them.

Each push each other apart but all of them are pulled towards the center of the viewport.

  • Click on a node to fix it. (ignores forces)
  • Click+drag to move the node around

Edit Mode

Edit the graph connections, node values, connection weights or delete nodes and connections

  • Click on empty area to add a new node
  • Click on a node to start a new connection
    • Click on an unconnected node to create a new connection
    • Click on a connected node to remove the connection
    • Click on an empty area to stop creating connections
  • Double-click a node to edit its value
  • Right-click a node to delete it
  • Double-click a connection to add a weight value (it is only visual/has no efect on simulation)
